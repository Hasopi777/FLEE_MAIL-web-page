How do you install our add-on?

An installation guide

  1. 1. In the google suite go to publish
  2. 2. Click deploy from manifest
  3. 3. Click get ID
  4. 4. Copy individual ID code
  5. 5. Open up gmail and login
  6. 6. Click the settings icon
  7. 7. Select settings
  8. 8. Open the gmail add-on settings tab
  9. 9. Click enable developer add-ons for my account
  10. 10. Paste the deployment key into the Developer add-ons input field: AKfycbw6TJxu_XSKw_f3p_iQQpLL-iHQ9sVGIj64M0ExjUhy
  11. 11. Click install
  12. 12. Refresh the page
  13. 13. Add-on should be displayed in the sidebar. You can now use it
